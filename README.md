# 💌 Vandy's Personal Website

## Introduction
Hi, I'm Vandy!

I'm a software engineer who almost never codes outside of work or school, so this is actually my **first personal project**.

Since there isn't really a social media platform that I feel comfortable posting on, I figured this could be my own little space on the internet instead. This website is where I'll keep the things that feel worth remembering — thoughts, feelings, moments, or anything I want my future self to look back on someday.

Visit vandyzhang.com or beta.vandyzhang.com if the website is currently in maintenance 😉

## Getting started

First, run the development server:

```bash
# format code
npm run format
# build
npm run build
# run it locally
npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
